Project Name: Nostra - E-commerce Website
-----------------------------------------

Description:
Nostra is a fully responsive e-commerce website built using HTML, CSS, and JavaScript. 
It includes three main pages – Home, Collections, and Contact Us. 
The Home page showcases trending products and popular fashion brands. 
The Collections page provides search and filter functionality using JavaScript 
to help users easily browse products by category. 
The Contact page allows users to get in touch with the store team. 
All pages are designed with a clean and modern layout for an enhanced shopping experience.
